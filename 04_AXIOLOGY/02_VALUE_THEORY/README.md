raises the individual symbiont and the sustaining holobiont together:


the shared field from extractive individual action. Due process keeps either
side from calling its own appetite "justice." The practical axiology test is
therefore: disclose the value claim, test it, ask whom it serves, preserve
Beauty/Truth/Justice, dissolve hidden rent, leave receipts, and return.

In the action register, `V` means usable D4 means-to-act: body, tools, energy,
access, execution capacity. `Φ` means actual D4 model capacity: tokens and
ranking/selection events that represent D5 possible content. Objective morals
are the `i -> H` vector that raises the holobiont; objective ethics is the
`H -> i` vector that raises the symbiont; strict syntropy is both focal
directions rising together under complete Justice, with all other affected
bearers nonnegative.

For the complete materially affected set `A`, use the commitment receipt `q`,
world-issued outcome receipt `r`, retrospective Justice `J^R(q,r;A)`, and
receipted durable changes `Δ^R W_b(q,r)`:

