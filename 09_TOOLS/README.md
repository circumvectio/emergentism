# Emergentism instruments

This lane contains repository utilities, deterministic renderers, research
simulations, and validation scripts for Emergentism alone.

## Authority rule

- Tools may compile, lint, render, test, and summarize.
- Generated output never becomes doctrine merely because a program emitted it.
- When generated output conflicts with an owner document, repair the owner or
  generator explicitly and leave a receipt.
- Ordinary AI work follows the user's stated scope and the permissions of the
  tools being used. Consequential action additionally requires an accountable
  authorization envelope naming principal, mandate, scope, consent, custody,
  contest path, actor, and consequence bearer.
- Private money or contract signatures are external legal acts, not a work gate
  and not an Emergentist primitive.

## Active surfaces

- `01_SCRIPTS/` — register, link, corpus, and research utilities.
- `02_COMPILERS/` — deterministic Burri plate renderer and semantic tests.
- `03_SIMULATIONS/` — mathematical and scientific models downstream of canon.
- `04_DATA_PIPELINES/` — current non-retired research pipelines.
- `07_AGENT_OPS/generate_agents_md.py` — deterministic pure-Emergentism route-card generator.
- `08_AUDIT_ARTIFACTS/` — current generated audit outputs only.
- `90_ARCHIVE/` — cold tool history, never an active command surface.
