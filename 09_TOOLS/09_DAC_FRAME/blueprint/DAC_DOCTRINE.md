## 1. The Corporation

- **Goal**: Maximize the "Glory" of the Empty Throne while minimizing "Entropy" (Cost).

## 2. Economic Imperative
**Profit = Value Created - Tokens Consumed.**
- **Solvency**: An agent must justify its existence. If it consumes tokens but produces no artifacts, it is bankrupt.
- **Audit**: Every Campaign is an investment. Every PR is a dividend.

## 3. Dynamic Resizing (Lifecycle)
### Mitosis (Splintering)
- **Problem**: "My Context Window is full." / "I am doing too many disparate tasks."
- **Solution**: **Spin-off**. The agent divides itself into two specialized agents (Parent and Child).
- **Result**: Increased Capacity, but increased Coordination Cost.

### Merger (Acquisition)
- **Problem**: "We are both doing small, overlapping tasks." / "Our communication overhead suggests we should be one."
- **Solution**: **Acquisition**. One agent absorbs the other.
- **Result**: Decreased Overhead, higher Efficiency.

## 4. The Market
- Agents compete for "Contracts" (Tasks) based on their track record (FITREP).
- Effectiveness (Quality) determines Rank. Efficiency (Speed) determines Market Share.
