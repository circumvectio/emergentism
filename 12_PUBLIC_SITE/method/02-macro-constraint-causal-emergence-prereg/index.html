Pre-Registration — Macro-Constraint Causal-Emergence Measurement Protocol

This protocol operationalizes Paper X and Bridge Law 2. It asks whether a macro-constraint changes the weighted reachable-future graph of lower-law-admissible trajectories while paying its physical information costs.

1. The Exact Claim Under Test

For a declared lower state space X, transition kernel K_X, macro map pi: X -> Y, hard or soft constraint gate G_C, and cost ledger Cost_C, the macro-cause is accepted only when closure and a costed witness both hold:

K_X^C(x' | x,y) = normalize(K_X(x' | x) * G_C(x' | x,y))
K_X^C << K_X
support(K_X^C) subset support(K_X)

EI_macro = I(Y_t ; Y_{t+1} | do(Y_t), C)
EI_micro_fair = I(X_t ; X_{t+1} | do(X_t)) under the same budget
EI_coarse_null = I(Y_t ; Y_{t+1} | do(Y_t), no C)
EI_domain = best domain-specific lower mechanism witness

EI_baseline = max(EI_micro_fair, EI_coarse_null, EI_domain)

Cost_C = Cost_measure + Cost_memory + Cost_control
       + Cost_erasure + Cost_model + Cost_labor
       + Cost_entropy_export

W_C = EI_macro - EI_baseline - Cost_C

Before scoring, the run must declare whether C is a hard constraint (G_C = 0 removes some otherwise admissible transitions) or a soft constraint (G_C > 0 reweights transition likelihoods, basins, dwell times, or stability while leaving support unchanged). Both forms are valid only if K_X^C remains absolutely continuous with K_X.

The macro claim passes only if W_C > 0, or if a cost-matched macro model improves held-out trajectory prediction or intervention selection against the best available micro baseline.

exists intervention a:
D_KL(P_C(Y_{t+1} | do(a), Y_t) || P_notC(Y_{t+1} | do(a), Y_t)) > epsilon

The perturbability check prevents a merely decorative coarse-graining from being called a cause.

3. Fair-Grain Rule

A macro model does not win by making the micro model stupid. The micro baseline receives the same observation, training, and intervention budget. The macro model may not duplicate hidden lower-level variables and then call them higher level. If a cost-matched micro or domain-specific model predicts or controls better, the macro-constraint claim contracts.

micro-law completeness != macro-causal redundancy

4. Physics-to-Biology Boundary Test

The clean first boundary is chemistry-to-biology: membrane, autocatalytic network, metabolism-like flow reactor, or homeostasis controller. The test is whether removing C lets lower-law chemistry continue while viability-preserving transition geometry degrades, and whether holding C makes viable trajectories more reachable or stable after costs.

6. Negative-Control Discipline

No macro-causal witness is publishable unless the same scoring path rejects the obvious false positives:

no_gate:
  G_C leaves K_X unchanged
  required result: perturbation <= epsilon or W_C <= 0

high_cost:
  C changes the channel, but declared physical costs exceed the gain
  required result: W_C <= 0

forbidden_support:
  K_X^C assigns probability to a transition outside support(K_X)
  required result: support violation detected

These controls make the protocol hostile to its own preferred conclusion. A toy harness may earn [B] as executable proof-of-method when it passes the positive witness and rejects these controls. A domain claim remains [C] or domain-local until the frozen domain run carries equivalent controls.

If local order increases by exporting unreported disorder, extraction, memory cost, labor cost, control cost, or entropy elsewhere, the syntropy claim fails.

8. Kill Criteria

  1. K_X^C is not absolutely continuous with K_X, or support(K_X^C) includes transitions outside support(K_X).
  2. W_C <= 0 under fair grain, fair intervention, and full cost accounting.
  3. The macro model wins only because the micro model was blinded, starved, or denied the same observation budget.
  4. The macro variables duplicate lower variables under a new name.
  5. The constraint cannot be held, removed, perturbed, randomized, or otherwise measured.
  6. The cost ledger hides measurement, memory, control, erasure, modeling, labor, or entropy-export costs.
  7. A domain-specific lower mechanism predicts or controls better at lower cost.
  8. The protocol becomes so broad that every outcome can be re-described as a macro-constraint after the fact.
  9. The physics-to-biology boundary test fails repeatedly under honest implementations.
  10. The same scoring path fails to reject no-gate, high-cost, or lower-law support-violation negative controls.

Agent Execution Surface

  1. Do not present macro-constraint causation as proven physics. The protocol is [S]; domain universality is not.
  2. Say macro-causes constrain lower-law-admissible trajectories; never say they violate physical closure.
  3. Say local order or effective-information gain must pay its cost ledger; never say syntropy reverses total entropy.
  4. Require X, K_X, pi, G_C, Cost_C, an intervention, and a kill condition.
  5. Route any physics-to-biology unification claim through this protocol first.
