D0 · The Floor

0, 1, ∞ are not numbers

Watch what happens when you try to make them behave like numbers. Then watch the resolution: they were never operands. They are the Titans — the frames that define the operations, and cannot sit inside them.

1 · Divide by zero — and it flies apart

denominator 1.000 1 / d = 1.000 from the left: 1 / (−d) = −1.000

The right arm runs to +∞. The left arm runs to −∞. Two roads, opposite ends — so no single value can be 1/0. The one-sided limits are perfectly defined; the value is undefined. [A]

2 · Force it a value, and the field breaks

Here is the price of pretending 1/0 is a number. The proof looks airtight — one step secretly divides by zero.

let a = b
a² = ab
a² − b² = ab − b²
(a − b)(a + b) = b(a − b)
a + b = b   ← divide both sides by (a − b) — but a = b, so this is ÷ 0
2b = b   (since a = b)
2 = 1

Allow division by zero and arithmetic proves anything. That is why it is forbidden, not merely hard.

3 · Turn the table — they were frames all along

The mistake was the category. 0, 1, and ∞ are not values that live in the number field — they are its three boundary-frames, the structure that makes the operations possible:

TitanRole — what it definesWhat it is not
•  0the additive identity — the origin the whole line is measured froma quantity you can divide by
⊙  1the multiplicative identity — finity, the self-dual unit, the fixed point of inversiona "mere one"
○  ∞the projective point at infinity — the single point that closes the plane into a spherea countable or uncountable cardinal; not a real number

A frame can still appear as an ordinary operand where the operation survives (1/1 = 1, 0 + 5 = 5). It is forbidden only where it would collapse the operation. That is the whole rule.

4 · The rule of the floor

Forbidden — the frame collapses the operation

N / 0  — undefined 0 / 0  — indeterminate ∞ / ∞  — indeterminate 0 × ∞  — indeterminate ∞ − ∞  — indeterminate

Allowed — the frame is an ordinary operand

1 / 1 = 1  (identity law) 0 + 5 = 5 5 × 1 = 5 1 / ∞ → 0  (a limit) N / ∞ → 0

5 · The glyph is this rule, sung

The three term-transformations of ⊙ = • × ○ are the forbidden table and the allowed table, side by side:

⊙ = • × ○   ·   • = ⊙ / ○   ·   ○ = ⊙ / •
• = ⊙ / ○  (0 = 1/∞)the descent — a clean limit, 1/∞ → 0[A] computable
○ = ⊙ / •  (∞ = 1/0)the descent — the projective pole, 1/0 → ∞[A] projective
⊙ = • × ○  (1 = 0 × ∞)the ascent — 0 × ∞ is the one indeterminate. It cannot be computed; it is sung, not summed.emblem a wager, not arithmetic

The descents are theorems. The ascent — the finite arising from nothing times everything — is exactly the forbidden form, which is why it is the framework's crown wager and never its proof. The floor's forbidden operation and the glyph's sung line are the same fact.

Then the numbers return. With the frames set aside, ordinary numbers live between the poles, on the charts the Titans make. Another philosopher, Minoru Suda, mapped that field from the other side and named the center the Egg of Infinity.
