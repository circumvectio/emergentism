0. Axiomatic guard

The framework claims that mirror symmetry around L4 is geometric necessity, not aesthetic preference. On S² with φ = cot(θ/2) and ν = tan(θ/2), every L-level at colatitude θ has a unique partner at the complementary angle (90° − θ) where the φ/ν roles are reversed. The pole-meeting (L0=L∞) closes the topology.

This means: for any column claiming to be a 7-projection of S², the L1↔L7, L2↔L6, L3↔L5 pairs must show functional inversion — same kind, opposite direction. If a column passes this test cleanly, it's a genuine S² projection. If it fails, it's a forced fit or a linear domain pasted onto a circular geometry.

This document operationalizes that test. Stronger than evidence-tier annotation alone: instead of marking a column "[S]" or "[S]," we score each MIRROR PAIR within each column. A column with 3 strong mirrors is doing real geometric work. A column with 3 fail-grade mirrors is decorative.

Zero-Sum Resolution Equation

1. The protocol

For each 7-row column in the Rosetta, score the three mirror pairs:

  • L1 ↔ L7: Boundary mirror (near-pole positions). The pair should show same kind of boundary pressure, inverse direction. L1 = pre-social raw threat (boundary from below); L7 = post-social transcendent closure (boundary from above).
  • L2 ↔ L6: Half-balance mirror (B = 0.5). The pair should show same balance position, inverse mode. L2 = first-contract / gathering-in; L6 = renunciation / refusing-out.
  • L3 ↔ L5: High-balance mirror (B = √3/2). The pair should show same high-balance position, inverse modality. L3 = productive-active building; L5 = contemplative-receptive holding.

Scoring:

Score Meaning
✓ STRONG Functional inversion is immediate and obvious without framework glasses. A naive reader would notice the symmetry.
◐ PARTIAL Mirror holds with framework reading; not obvious to an outsider; defensible but interpretive.
✗ FAIL No functional mirror. The L5/L6/L7 entries seem linearly higher than L3/L2/L1, not reflectively paired.

Per-column FALSIFICATION SCORE: count the failures across the 3 pairs. - 0 fails (all STRONG/PARTIAL): column is a genuine S² projection - 1 fail: column is mostly clean but has a structural weak point - 2 fails: column is partially forced - 3 fails: column is linear, not reflective; treat as decorative

Anumāna = knowing-from-premise (deductive); Śabda = knowing-from-authority (received testimony). Same mode (mediated propositional), inverse source (self-derived vs other-given).

Falsification score: 0 fails. Pramāṇa column is a genuine S² projection. This is the gold standard.

The mirror holds at the structural level (rule-by-elite) but the inversion direction is more "moral upgrade" than "geometric reflection."

4.2 A defense against the column-taxonomy critique

Pressure Point #1 says "15 columns demand explanation, but 7 are derived." Mirror symmetry test cuts orthogonal to this: a derived column can still pass or fail the mirror test. Derivation doesn't pre-determine score; geometric structure does.

The Pramāṇa column is internally generated (Vedāntic, traditional), but it scores 3-strong. The Computation column is also internally generated (CS-traditional), but it scores 2-fail. The TEST is independent of derivation — it's about whether the column actually projects S² geometry or just mimics it.

4.4 A framework-improvement loop

When a column fails the mirror test, the test itself suggests two paths: 1. Accept the column as decorative. Mark it [C] conjectural; remove from canonical V-bis. 2. Re-derive the L5/L6/L7 entries from the mirror principle. Force the framework to PRODUCE the missing inverse for each L1/L2/L3 entry.

Path 2 is genuinely generative: the framework must EARN its L5-L7 entries by mirror-completion of L1-L3. If a domain has clean L1-L3 (which most domains do, because most domains have low-boundary, half-balance, and high-balance entries), the framework predicts the L5-L7 entries by inversion. If those predictions match the source domain, the framework wins. If not, the column is decorative.
