0. Column Ceiling

Neuroscience is the most testable and most dangerous column. L1-L4 facts are comparatively grounded, but the L5-L7 mapping is interpretive or conjectural. The current column reads partly as a linear cognitive sophistication ladder, not a mirror-complete S² projection.

This packet preserves both:

  1. Current column: the Master Rosetta entries.
  2. Mirror-completion alternate: the predicted L5-L7 entries generated by inverting L3-L1.

The test is empirical: if the mirror-completion alternate maps better to neuroscience literature than the current upper half, update the column. If neither maps, demote neuroscience.

5. Failed / Partial Appendix

  • Current upper half is linear. L5-L7 currently read as more cognitive sophistication rather than mirror inversion.
  • Hemisphere claims must stay tendencies. Hemisphere = tendency, never identity.
  • DMN is not witness by default. Ordinary DMN is often self-narrative; witness requires reconfiguration or a different network claim.
  • L7 is conjectural. It must never be public evidence for the Rosetta until independently supported.
