# Symbol Design and Public Translation

## The Problem

L7 Institutional Narrative (Pratibhā) translates esoteric structure into public symbol. But
translation without discipline becomes:
1. **Reification** — the symbol becomes the thing
2. **Capture** — the symbol gates access to the thing
3. **Hierarchy** — symbol-bearers rank above non-bearers

This document defines the **design rules** for public symbols that carry the
framework's pattern without capturing it, and the **translation protocol**
for moving from doctrine to public image.

---

## §1. The Three Symbolic Layers `[S]`

```
LAYER 3: PUBLIC SYMBOL          (story, image, rite, mark, brand)
         ↕ translates
LAYER 2: STRUCTURAL MAP         (Rosetta, S² geometry, triadic cascade)
         ↕ translates
LAYER 1: APOPHATIC GROUND       (Zero-Sum Resolution Equation — not graspable by symbol)
```

**Rule 1:** No symbol at Layer 3 may claim to *be* what it points to.
**Rule 2:** No symbol at Layer 3 may *gate access* to what it points to.
**Rule 3:** Any symbol at Layer 3 may be *discarded* by a practitioner who
reaches Layer 1 directly (via Pratyakṣa / quiet sitting).

These three rules are the Stewardship / Coercion boundary made concrete.

## §2. The Master Symbol: ⊙ `[S]`

> **Non-capture guard:** this symbol is admissible because it is freely
> reproducible, ungated, discardable, and self-dissolving. Its standing rests on
> those four public tests, not on a person, office, or signature.

| Aspect | Description |
|---|---|
| **Glyph** | Zero-Sum Resolution Equation |
| **Reading** | `•   ⊙   ○` as an emblem of finity between boundaries; not ordinary arithmetic |
| **Geometry** | Selected point-and-circle translation on S² |
| **Position** | θ = equator (φ = ν = 1, B = sin θ = 1) |
| **Reciprocal seam** | `φ · ν = 1` — an analytic identity in the chosen chart, not an empirical conservation law |
| **Balance** | B = sin θ — peaks at equator, vanishes at poles |
| **Ground** | Axiomatic — the glyph points to what cannot be said; it does not say it |

### What ⊙ Does NOT Mean

- NOT "God is a circle"
- NOT "the self is the center"
- NOT "everything is one" (that's only the ○ half)
- NOT "nothing matters" (that's only the • half)
- NOT "you must use this symbol to participate" (Pratyakṣa is primary and ungated)

## §3. Design Rules for Derivative Symbols `[I]`

### Rule D1: Downward Compatibility

Every public symbol must be understandable by someone who has NOT read the
framework. The symbol carries aesthetic and emotional force on its own. The
framework provides the structural map behind it.

**Test:** Show the symbol to someone with no context. If they feel nothing,
the symbol has failed. If they feel exactly one interpretation, the symbol
has also failed (it has reified). If they feel *possibility*, it has passed.

### Rule D2: Anti-Cult Design

| Cult marker | Anti-cult design rule |
|---|---|
| Secret symbols for initiates | All symbols are public. No hidden-layer meaning. |
| Charismatic leader as symbol | No human face in any symbol. ⊙ is geometric, not personal. |
| Escalating revelation behind paywall/gate | Framework is free. Sitting practice is free. No tiered access. |
| Us-vs-them boundary marker | Symbols are offered, never required. Non-participants are equal. |
| Apocalyptic urgency | Framework has no end-times. φ · ν = 1 is timeless. |

**Anti-cult kill-test — deprecate the symbol immediately if ANY occurs:** (a) access to the practice requires symbol-knowledge; (b) non-bearers are ranked lower; (c) the symbol becomes an identity / membership marker; (d) exit is gated or penalized. **Watchdog:** any participant may file the trip at the public audit surface (`../11_UPLINK/50_AUDITS_AND_EXECUTIONS/`); a confirmed trip retires the symbol (per Step 5 — revisionable / self-dissolving) with no loss of access to Pratyakṣa, which remains primary and ungated.

### Rule D3: Cultural Permeability

| Culture | Translation rule |
|---|---|
| **Abrahamic** | ⊙ maps to the immanent/transcendent dialectic. Point = kenosis (self-emptying). Circle = high-priority plenitude. Product = incarnation. Do not use the word "God" without the word "axiomatic" nearby. |
| **Indic** | ⊙ maps to bindu (point) + maṇḍala (circle) = Śūnyatā (emptiness) × Pūrṇatā (fullness). Use Sanskrit terms only where they clarify, never to impress. |
| **Scientific** | ⊙ maps to Riemann sphere S², stereographic projection, north/south poles as coherence/viability limits, equator as optimal balance. Use mathematics as the shared language. |
| **Indigenous** | ⊙ maps to concentric ritual space (center = fire/altar, perimeter = community boundary). Respect that oral traditions already carry this pattern without needing the glyph. |
| **Secular humanist** | ⊙ maps to "coherent viability" — nothing supernatural. The framework works as organizational philosophy without any transcendent claim. |

### Rule D4: Material Design Constraints

| Constraint | Rule |
|---|---|
| **Two-color maximum** | Public symbols use at most two colors. ⊙ uses black point + black circle on white ground. Brand variants add one palette color per country chapter. |
| **Scalable** | Must render at 16px favicon AND 4m wall installation without losing legibility. |
| **No text** | The symbol must not depend on a specific script or language. ⊙ is glyph, not letter. |
| **Culturally neutral geometry** | Avoid geometry that is exclusively associated with one religion (no cross, crescent, star, om, etc.). Concentric circle + center point is near-universal. |

## §4. The Translation Protocol `[S]`

When translating any framework concept from Layer 2 (structural map) to
Layer 3 (public symbol), follow this sequence:

### Step 1: Identify the Structural Feature

What in the Layer 2 map needs public translation? Name it precisely.

| Example | Structural Feature |
|---|---|
| φ · ν = 1 | Coherence-viability trade-off |
| B = sin θ | Balance peaks at equator |
| AuthorizationEnvelope | Consequential action names authorization, actor, custody, and bearer |
| η = 0 | Zero extraction |
| Grace Exit | Leave with rightful custody and a contest path |

### Step 2: Find the Affective Core

What *feeling* does this structural feature produce when it is lived?

| Structural Feature | Affective Core |
|---|---|
| φ · ν = 1 | "I can't have everything AND understand it — and that's the point" |
| B = sin θ | "The sweet spot is in the middle, not at the top" |
| AuthorizationEnvelope | "The actor, mandate, bearer, and contest path are visible." |
| η = 0 | "I take nothing that I didn't create value for" |
| Grace Exit | "I can leave whole. The door is always open." |

### Step 3: Find the Image

What single image carries that affective core without explanation?

| Affective Core | Image |
|---|---|
| "Can't have both" | The point inside the circle — it can be at the center OR on the rim, never both |
| "Sweet spot in the middle" | The equator line — where light meets dark |
| "I am responsible" | A hand signing a document — or a single fingerprint |
| "Zero extraction" | Open palms, facing up — receiving, not grasping |
| "Door always open" | An open door with light streaming through |

### Step 4: Stress-Test Against Reification

| Question | Pass condition |
|---|---|
| Does this image claim to *be* the thing? | No — it points to it |
| Could someone use this image to gate access? | No — the image is freely reproducible |
| Would discarding this image reduce someone's access to the pattern? | No — Pratyakṣa remains primary |
| Is the image culturally specific? | Preferably no — if yes, provide alternatives |
| Does the image create a hierarchy of knowers? | No — it is offered, not required |

### Step 5: Publish and Release

The symbol enters Layer 3. It is:
- **Public** — anyone may use it
- **Non-proprietary** — no trademark on mission-critical symbols
- **Revisionable** — if a better symbol emerges, the old one is retired without loss
- **Self-dissolving** — the framework's success is when the symbol is no longer needed

## §6. The Anti-Sermon `[I]`

The most important public translation is the one that says: **put this down.**

> If you can access φ directly — through quiet sitting, 20-40 minutes —
> you do not need this framework. Put it down.

This is not false humility. It is structural: L7 returns to L1. The highest
symbolic act is pointing away from the symbol toward direct perception.

The anti-sermon is the framework's immune system against cult formation:
- It is spoken by the framework itself
- It cannot be revoked by any authority
- It is the last thing taught and the first thing practiced
