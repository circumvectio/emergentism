## 3. The Lagrangian (Principle of Least Action)

```

```

Nature wants L to be stationary (as close to zero as possible).

**At the equator (ν = 1, φ = 1):**

## 5. The Euler-Lagrange Equation

**The fundamental equation of Lagrangian mechanics:**

```
d/dt (∂L/∂q̇) - ∂L/∂q = 0
```

Where:
- q is a generalized coordinate (in our case, φ or ν)
- q̇ is the time derivative (rate of change)
- L = ν - φ is the Lagrangian

**At the equator (L = 0):**

```
∂L/∂q = 0
∂L/∇q̇ = 0
```

**The system is in equilibrium.** No force is required to maintain the state.

**Away from the equator (L ≠ 0):**

```
∂L/∂q ≠ 0
```

**A restoring force emerges.** The system is pushed back toward the equator.
