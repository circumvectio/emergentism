# The Coherent Geometry

This is a compact lens, not a proof that reality is one manifold.

## 1 · Reciprocal chart `[A]`

For `θ∈(0,π)` define

```text
φ = cot(θ/2) > 0
ν = tan(θ/2) > 0
B = sin θ
```

Then

```text
φν = 1
φ + ν ≥ 2
B = 2/(φ+ν) ≤ 1
```

Equality in the last two relations occurs only at `θ=π/2`, where
`φ=ν=1`. These are analytic facts of the selected chart. They are not an
empirical conservation law, ontology, ethic, or evidence for a finite-node
aggregator. The poles are limits; `0·∞=1` is not field arithmetic.

## 3 · Relativistic correspondence `[A/I]`

For positive rest mass and one signed momentum component, one may set

```text

ν = (E−pc)/(mc²)
```

ethic. The formula is not defined for a photon by dividing through `m=0`.
