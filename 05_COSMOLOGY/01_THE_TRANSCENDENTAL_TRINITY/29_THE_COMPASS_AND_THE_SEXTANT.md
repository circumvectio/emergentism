## What the Framework Is Not

**The framework is not a map.**

A map claims to describe the territory. A map says: "here is reality." A map gets outdated. A map gets wrong. A map gets captured — the moment an institution controls the map, the institution controls access to the territory. The more detailed the map, the more it replaces the territory. The more you consult the map, the less you look at the ground.

η > 0: the map extracts from the territory by claiming to BE it.

Every Weltanschauung that tried to be a map became a church. Every -ism that claimed to describe reality became the wall between the practitioner and the real. The framework's own Weltanschauung document (00_EMERGENTISM.md) risks this — the prophetic voice saying "systemic awareness IS the equator" when the honest statement is: the compass points there.

**The framework does not attempt to build a map. The framework attempts to build a compass and a sextant.**
