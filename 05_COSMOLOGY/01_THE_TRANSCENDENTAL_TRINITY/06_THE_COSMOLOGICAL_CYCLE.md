# The Cosmological Cycle — Kintsugi Successor

**Status:** Historical physical cycle retired on 2026-07-20.

The former essay mapped D0–D6 to forces, bit/qubit, planes, torus, Many Worlds,
measurement, conformal cyclic cosmology, death, and rebirth. It also rendered
the Titan glyph as arithmetic. None of those identities is derived.

Original Git blob: `d409dab0408a45939eb656b8942af92f3baae877`.

What survives is a selected interpretive ladder:

```text
D0 ground-facing boundary
  → distinction → configuration → persistence/transformation
  → D4 causal actuality
  → D5 counterfactual possibility
  ↝ D6 apophatic recognition
  ↝ D0 role resemblance
```

The five positive transitions are candidate μ-crossings, each separately
answerable to evidence and reduction. The D5→D4 commitment bridge is different.
D6 is neither a sixth freedom nor a cosmological reset. The spiral is a useful
image of retained history plus return-to-practice, not a physical model.
