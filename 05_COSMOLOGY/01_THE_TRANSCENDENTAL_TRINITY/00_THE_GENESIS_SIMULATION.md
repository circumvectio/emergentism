The simulation passes only when:

1. it renders exactly `D0…D6`, `μ₀…μ₄`, one `b₆`, and one `r₆`;
2. D4 elements are actual and D5 elements possible;
3. no frame claims a point produces a sphere or numbers;
4. no caption says computation proves ontology, emergence, consciousness,
   entropy convergence, quantum collapse, or cosmological recurrence;
5. deleting quantum-interpretation commentary changes no D3 calculation or
   operational rule;
6. deterministic input produces deterministic output bytes.
