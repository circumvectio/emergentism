# Saturation, Contrast, and Aperture

This note isolates four claims that had begun to look like one:

1. an exact reciprocal-chart limit;
2. an exact result inside a separately budgeted node model;
3. an exact local gauge-theory example;
4. a conjecture that saturation of one capacity can extinguish the contrast or
   complementary capacity required for action.

Only the first three have theorem-shaped content. The fourth is a research
heuristic and cannot be used to establish a mu crossing.

## 1. Reciprocal-chart boundary `[A]`

For `theta in (0,pi)`, define

```text
phi = cot(theta/2)
nu  = tan(theta/2)
B   = 2/(phi+nu) = sin(theta).
```

Then

```text
phi*nu = 1,
phi -> infinity  =>  nu -> 0 and B -> 0,
nu  -> infinity  =>  phi -> 0 and B -> 0.
```

These are analytic facts of the selected chart. They imply no resource budget,
physical conservation law, ontology, ethic, force, or emergence event.

## 2. Budgeted node boundary `[A/S]`

Let `Phi,V in [0,1]`, let `P_node=Phi*V`, and separately impose

```text
Phi + V <= 1.
```

Then:

```text
Phi = 1  =>  V = 0  =>  P_node = 0,
V = 1    =>  Phi = 0 =>  P_node = 0,
P_node <= 1/4,
P_node = 1/4 iff Phi = V = 1/2 under Phi+V=1.
```

The proof of the maximum is AM-GM or completion of the square:

```text
Phi*V <= ((Phi+V)/2)^2 <= 1/4.
```

The interpretation is conditional: exhaustive allocation to represented
foresight leaves no declared means, while exhaustive allocation to means leaves
no declared foresight. The two poles are operationally sterile only inside the
named budget model.

### Countermodel: the product supplies no tradeoff

Remove `Phi+V<=1`. The admissible assignment `Phi=1,V=1` gives `P_node=1`.
Therefore neither the product nor the reciprocal chart proves that foresight
must consume means. A real application owes independent measures, an observed
resource surface, and comparison with rival constraints.

## 3. Local gauge contrast `[A/I]`

For an Abelian gauge potential one-form `A`, the local field strength is

```text
F = dA,
F_mu_nu = partial_mu A_nu - partial_nu A_mu.
```

If `A=d lambda` locally, then

```text
F = d(d lambda) = 0.
```

A constant potential is one local special case. The exact statement is:

> A locally pure-gauge potential has zero local field strength.

This is a statement about gauge-equivalent description and vanishing local
curvature, not about an infinitely strong electromagnetic field becoming no
field. A nonzero constant field strength remains nonzero. On a multiply
connected region, locally vanishing `F` need not remove gauge-invariant global
holonomy; the Aharonov-Bohm case blocks that inflation. See the
[gauge-independent analysis](https://arxiv.org/abs/2209.11091).

## 4. The Saturation-Contrast conjecture `[C]`

The three results above motivate, but do not prove, this domain-general wager:

> In some finite systems, pushing a representational or control capacity to its
> declared resource boundary can eliminate the complementary capacity,
> difference, gradient, or contrast through which that capacity has an
> observable effect.

Call this `SC-CANDIDATE`. It is not an axiom, universal law, or definition of
mu. It is a sub-wager of W4 and a candidate mechanism to test at particular
crossings.

### Required test packet

Every proposed instance must declare:

```text
systemBoundary
capacityX
complementY
resourceSurface
observableEffect
saturationStatistic
saturationThreshold
intervention
strongestRival
heldOutPrediction
killCriterion
```

The minimum discriminator compares at least:

1. a no-tradeoff model;
2. an additive budget;
3. one nonlinear or state-dependent resource surface;
4. a model in which the observable depends on gradient or contrast but not on
   saturation.

### Kill criteria

The local instance dies if:

- `X` can saturate without reducing `Y` or the observable effect;
- the claimed loss is explained by an already measured bottleneck;
- the saturation statistic was chosen after seeing the alleged aperture;
- another resource surface predicts held-out observations as well;
- the effect depends on an untyped change of equivalence relation; or
- the analogy substitutes for a mechanism.

The global generalization dies if well-powered, preregistered instances across
the frozen domain set show no recurrent structure beyond ordinary resource
competition, gauge equivalence, or diminishing returns.

## 5. Relation to mu

`SC-CANDIDATE` can at most propose a trigger variable for `mu_1...mu_4`. A mu
claim still owes all fields in the canonical crossing record: source register,
new freedom, lower-register recovery, reduction status, evidence tier,
prediction, and kill. In particular:

- `mu_0` is an origin aperture and has no prior positive saturation;
- formal enlargement or re-description is not emergence;
- loss of an old observable is not by itself appearance of a new freedom;
- a successful lower-level reduction reclassifies the crossing rather than
  destroying the dimension-first address system.

## 6. Compact verdict

```text
reciprocal extremity -> loss of chart balance                    [A]
budget extremity     -> loss of complementary node capacity     [A/S, conditional]
pure gauge           -> zero local field strength               [A, local]
universal saturation -> aperture / force cancellation           [C, unearned]
```

The beauty is the recurrence. The truth is the type boundary. The just use is
to test the recurrence without forcing the world to complete the pattern.
