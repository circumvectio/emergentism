# Emergentism Public Site Boundary

**Date:** 2026-06-08
**Public surface:** `12_PUBLIC_SITE/`

## Boundary

`12_PUBLIC_SITE/` is the public web surface. It is source-bearing and remains in
the root repository, but generated dependencies and build output are excluded.

## Preservation Rule

- Keep source routes, rendered static/source docs, assets, scripts, package
  manifests, and lockfiles.
- Delete only regenerable caches and dependency/build folders.
- Do not treat publication to GitHub as a claim upgrade for any doctrine,
  experiment, or public-facing statement.
