The historical name is retained for path stability. The repaired program does
not assign computation, measurement, ranking, selection, or thermodynamic cost
to D5. Those are actual D4 processes. D5 names only the possible alternatives
represented as content by an actual D4 model token.

## Operational bridge

field `Ω_t`; let `V_t` and `U_t` be actual means and authorization. An actual D4
selection-and-commitment operation is typed as:

```text

```

`q_t` is the D4 commitment receipt. It records what was authorized and
committed; it is not the consequence. The environment separately returns:

```text

```

`r_{t+1}` is the D4 outcome receipt. This separation prevents the selector from
manufacturing its own success.
