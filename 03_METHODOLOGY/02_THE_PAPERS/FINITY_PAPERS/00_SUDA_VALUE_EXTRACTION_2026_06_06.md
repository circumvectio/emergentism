## The Core Extraction

### 1. Division as double inversion

Suda's Part I frames ordinary division as a two-part switch:

```text
operator: division -> multiplication
operand:  y -> y^-1
```

This is useful because it makes division visibly relational: division is not merely an operation on a number; it is a change of action and a change of perspective. In Emergentism terms, this helps explain why division by zero cannot be repaired by pretending the same register still applies. At the boundary, the operation is asking for a chart change.

Public formulation:

> Division is already a hidden inversion. At zero, the hidden chart change becomes visible.

### 2. Reciprocal geometry makes `1` unavoidable

On `R_+`, Suda's reciprocal map is:

```text
I(x) = 1/x
```

Its fixed point is:

```text
x = 1
```

In logarithmic coordinate:

```text
s = log x
I: s -> -s
```

This makes `1` the mirror center, because `x = 1` corresponds to `s = 0`. Zero and infinity exchange under inversion; `1` does not exchange with another point. It returns to itself.

Emergentism adoption:

```text
• <-> ○ under inversion
⊙ fixes itself
```

This is the clean mathematical backbone for `/1`.

### 3. Energy around unity

Suda defines:

```text
rho(x) = |log x|
E(x) = (log x)^2
```

`E` is invariant under reciprocal inversion and is minimized uniquely at `x = 1`. This gives us a precise language for "distance from finity" without measuring from zero.

Important refinement for our doctrine:

- Additive arithmetic measures from `0`.
- Multiplicative/reciprocal geometry measures from `1`.
- The special position of `1` becomes obvious only after changing chart.

Public formulation:

> If you measure by addition, zero looks like the origin. If you measure by reciprocity, one is the origin.

### 4. The phase bit distinguishes the two sides

Part III adds:

```text
phi(x) = sign(log x)
```

This separates magnitude from side:

```text
E = strength or distance from unity
phi = which side of unity
```

For a response `F(x)`, Suda's even/odd split is:

```text
F_even(x) = 1/2 [F(x) + F(1/x)]
F_odd(x)  = 1/2 [F(x) - F(1/x)]
```

This is useful for Emergentism because it gives an operational grammar for mirror phenomena:

- even terms are invariant across the mirror;
- odd terms flip sign across the mirror;
- the mirror itself is `1`.

This can later become a tool/calculator design principle: make the mirror explicit instead of plotting everything as if zero were the only origin.

### 5. The egg coordinate

Suda's hinge coordinate is:

```text
u = (x - 1)/(x + 1)
x = (1 + u)/(1 - u)
```

It maps the positive ray into a bounded interval:

```text
x -> 0        gives u -> -1
x = 1        gives u = 0
x -> infinity gives u -> +1
```

Under reciprocal inversion:

```text
u -> -u
```

This is the clearest formal version of the "egg":

```text
zero pole      center       infinity pole
u = -1    <--  u = 0  -->   u = +1
0              1            infinity
```

Emergentism reading:

```text
• and ○ are the two rims of the egg.
⊙ is the center where the egg can be seen as one bounded thing.
```

### 6. Continuous half-twist

Part III turns the static inversion into a continuous path. With `u = sin(theta)`, a half-period produces:

```text
theta -> theta + pi
u -> -u
```

This matters for our animations and for Mu-limit language. It says the mirror operation is not only a symbolic swap; it can be visualized as a continuous half-twist through a projective coordinate.

Public formulation:

> The reciprocal does not jump from one pole to the other. In the right coordinate, it twists through the center.
