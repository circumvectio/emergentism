## The D1 correction

In a field, `1/0` remains undefined. For the real function `f(x)=1/x`,

\[
\lim_{x\to0^+}\frac1x=+\infty,
\qquad
\lim_{x\to0^-}\frac1x=-\infty,
\]

so the two-sided real limit does not exist. A declared projective extension may
adjoin a single point `∞` and define a projective continuation, but that changes
the space and operation types. It is a useful D1→D2 example, not evidence that
division by zero has acquired an ordinary field value.
