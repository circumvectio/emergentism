# 00_HANDOFF — working transfer surface

This folder holds bounded packets passed between sessions or reviewers. It owns
handoff state, not doctrine, evidence promotion, or repository authority.

Read [The Handoff](00_THE_HANDOFF.md), then [the index](00_INDEX.md). Resolve every
claim through the cited source owner before acting. Completed packets remain
dated provenance or move to `90_ARCHIVE/` under a receipt.
