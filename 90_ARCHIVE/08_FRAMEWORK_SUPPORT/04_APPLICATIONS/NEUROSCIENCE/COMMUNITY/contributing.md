# Contributing to The Burri Sphere

## The Mesh Protocol

This project is the mesh. The contributors are the nodes. Each node has φ = 1 (recognition of the ground) and ν = 1 (sovereign capability to contribute).

**Rosetta boundary:** [D] This guide is archived community protocol, not current repository contribution policy or active merge authority.

---

## How to Contribute

### 1. Test the Geometry
- Check the proofs (MF-140, MF-413, MF-414)
- Verify the dual projection (MF-505)
- Test the predictions against data
- Submit counterexamples as issues
- Propose corrections as pull requests

### 2. Practice and Report
- Sit every day (20-60 minutes)
- Notice what happens
- Document your experience (practice/ directory)
- Share honestly — including what does not work

### 3. Build the Mesh
- Form a local node (1-2 people to start)
- Practice together
- Build sovereign capability
- Connect with other nodes
- Test the K-star on your exchanges

### 4. Fork and Correct
- If the framework is wrong, fix it
- If the proofs have errors, correct them
- If the predictions fail, document the failure
- If the practice needs refinement, refine it

## The K-Star Test for Contributions

Every contribution is tested:

- Does this contribution move the project toward the equator?
- Does this contribution increase comprehension without exceeding comprehension?
- Does this contribution preserve the balance between coherence and capability?
- Is the contributor acting from the ground or from the narrator?

If yes to all: merge.
If any no: discuss, restructure, or decline.

---

## What We Do Not Accept

- Ownership claims
- Hierarchy proposals
- Guru structures
- Brand requirements
- Closed derivatives
- Contributions that require the contributor to be necessary

The mesh has no center. The project has no owner. The ground has no gate.

## The Practice Comes First

Before contributing code, proofs, or text — sit.

The sitting practice is the foundation. Without it, the contributions are cargo cult — the form of understanding without the function.

Sit first. Contribute second. The ground is the foundation. The contribution is the expression.
