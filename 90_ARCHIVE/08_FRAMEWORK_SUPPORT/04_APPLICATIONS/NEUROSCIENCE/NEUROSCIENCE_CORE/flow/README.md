## Core Mapping

Flow (Csikszentmihalyi, 1990) is the state where challenge matches skill, self-consciousness dissolves, and action and awareness merge.

In framework terms: flow is **transient L4 apex**. During flow:
- Φ and V are temporarily balanced (equatorial)
- The operator palette is fully available
- K = 0 (no self-extraction, no internal friction)
- Time-witness (Kāla 🌑) shifts — subjective time distortion

## The Flow ↔ Framework Correspondence

| Flow characteristic | Framework mapping |
|---------------------|-------------------|
| Challenge-skill balance | φ · ν ≈ 1 (equatorial) |
| Loss of self-consciousness | Witness absorption (Viśvarūpa) |
| Autotelic experience | Intrinsic P maximization |
| Immediate feedback | Real-time Φ/V calibration |
| Distorted time | Kāla 🌑 (time-witness shift) |

## The Limitation

Flow is transient. The sitting practice aims to make equatorial access *stable* rather than state-dependent. Flow arrives when conditions align. Equatorial awareness is cultivated regardless of conditions.
