## FRAMEWORK CONTEXT FOR REVIEWERS

This document is part of the **Emergentism** framework. The following definitions are provided so you can evaluate the enclosed predictions without any other document.

### Key Definitions

- **Burri Sphere:** The Riemann Sphere S² parameterized by dual stereographic coordinates φ (coherence) and ν (viability), where φ·ν = 1 everywhere.
- **φ (coherence):** Structural integration — how well a system's parts relate to its whole.
- **ν (viability):** Functional capability — what a system can actually do.
- **The equator:** The locus where φ = ν = 1. Maximum balance B = sin(θ) = 1.
- **B (Balance):** B = sin(θ) = 2ν/(1+ν²). The height function of S² embedded in ℝ³.
- **η:** Extraction coefficient. η = 0 means zero non-consensual transfer. η > 0 means structural parasitism.
- **The ontological claim:** φ·ν = 1 is maintained DYNAMICALLY — the product holds across disruption and recovery, at every scale, with hyperbolic (not sigmoidal) functional form.

### What Makes These Predictions Distinguishing

"The compensation follows the SPECIFIC hyperbola ν = 1/φ" is a prediction that only this framework makes. The experiments below test functional forms, not qualitative tendencies.

## PREDICTION 1: HYPERBOLIC COMPENSATION UNDER STRESS

### The Claim
When a system's structural coherence (φ) is suddenly disrupted, its functional capability (ν) increases to maintain the product φ·ν ≈ constant. The compensation is **hyperbolic** (ν = 1/φ), not sigmoidal (saturation at a maximum).

### Quantitative Form
$$\phi(t) \cdot \nu(t) \approx C \quad \text{(constant across the disruption-recovery cycle)}$$

### What This Predicts That Competing Theories Don't

| Framework Prediction | Standard Biology Prediction |
|---------------------|---------------------------|
| φ·ν product remains approximately constant | Product varies with disruption severity |
| Compensation curve is hyperbolic (unbounded as φ→0) | Compensation is sigmoidal (plateaus at system maximum) |
| Return path traces same hyperbola as disruption path | Proxies follow independent kinetics on return |
| System failure = required ν exceeds capacity to generate it | System failure = damage exceeds repair threshold |

### Proposed Protocol
- **System:** Cell cultures (epithelial monolayers or fibroblasts)
- **φ-proxy:** Structural integrity (membrane potential, cytoskeletal organization via impedance spectroscopy)
- **ν-proxy:** Metabolic rate (ATP production via luminescence, oxygen consumption rate)
- **Manipulation:** Graded mechanical disruption (increasing severity, 5+ levels)
- **Measurement:** Both proxies sampled at high temporal resolution through full disruption-and-recovery cycle
- **Analysis:** Fit product φ·ν across disruption range. Test hyperbolic vs. sigmoidal model fit.

### Kill Criterion
If the φ·ν product varies by more than 2 standard deviations across the disruption range (after calibration), the specific quantitative claim is falsified. The qualitative claim (compensation exists) survives, but the ontological form (hyperbolic, product-conserving) does not.

### Estimated Resources
Standard cell biology lab. Impedance measurement system (e.g., ECIS), metabolic assay (Seahorse or equivalent). 2-3 months.

### Priority: **RUN FIRST** — highest feasibility, highest distinguishing power.

## PREDICTION 2: CROSS-SPECIES SECURITY RATIO

### The Claim
Across species, the relationship between ecological security (freedom from predation) and physiological arousal follows the balance function B = 2ν/(1+ν²), producing a continuous curve — not a binary predator/prey step function.

### Quantitative Form
$$B_{\text{species}} = \frac{2\nu_{\text{security}}}{1 + \nu_{\text{security}}^2}$$

where ν_security is a composite measure of the organism's effective capability relative to environmental threat.

### What This Predicts That Competing Theories Don't

| Framework Prediction | Standard Ecology Prediction |
|---------------------|---------------------------|
| Continuous sin(θ)-shaped curve across species | Binary or categorical (predator vs prey) |
| Mesopredators show intermediate arousal matching the curve | Arousal depends on specific niche, no universal form |
| Within-species variation follows same curve (urban vs rural) | Within-species variation is genetically slow |

### Proposed Protocol
- **Species gradient:** Within shared ecosystem (e.g., African savanna): lion → leopard → wild dog → baboon → impala → hare
- **Security ratio:** Composite of body mass ratio to predators, predator density, cover availability
- **Arousal proxies:** Baseline cortisol (allometrically corrected), resting heart rate, sleep architecture (total hours, fragmentation index), startle reflex threshold
- **Analysis:** Fit arousal-vs-security data to sin(θ) curve, linear model, and step function. Compare model fits.

### Kill Criterion
If a step function or linear model fits the cross-species data significantly better (p < 0.05) than the sin(θ) form, the specific geometric claim is falsified.

### Estimated Resources
Field biology collaboration. Cortisol sampling across 6-8 species. 6-12 months.

## PREDICTION 3: RAPID EQUATORIAL RETURN AFTER PREDATOR REMOVAL

### The Claim
Removing chronic displacement forces (predation pressure) produces rapid behavioral and physiological shift toward the equator — within a single generation — at a rate proportional to the displacement magnitude.

### Quantitative Form
$$\frac{d\theta}{dt} \propto \nabla B = \cos(\theta)$$

Return rate is fastest far from the equator (large displacement) and slows as the system approaches balance.

### What This Predicts That Competing Theories Don't

| Framework Prediction | Standard Biology Prediction |
|---------------------|---------------------------|
| Rapid shift (weeks to months) — telos = equator | Slow shift (generations) — genetic drift |
| Return rate ∝ displacement magnitude | Rate determined by mutation rate and generation time |
| φ·ν product constant during return | Independent kinetic curves for each variable |

### Proposed Protocol
- **Natural experiments:** Island populations vs mainland populations of same species (Galápagos, Mauritius, Hawaii)
- **Controlled experiments:** Predator exclusion in field enclosures
- **Measure:** Time course of cortisol baseline, startle response, sleep architecture, exploration behavior
- **Critical test:** Speed — framework predicts within-generation change; standard biology predicts multi-generational

### Supporting Data Already Available
- Island tameness literature
- Predator removal experiments in ecological research
- Urban wildlife adaptation timelines

### Kill Criterion
If behavioral shift requires >10 generations after predator removal (matching genetic drift models), the rapid-return prediction fails.

## PREDICTION 4: PTSD AS FAILED EQUATORIAL RETURN

### The Claim
PTSD is what happens when a traumatic displacement exceeds the system's capacity for equatorial return — the system is stuck in a high-ν, low-φ configuration (hypervigilance + shattered coherence). Effective therapy = φ-restoration along the hyperbola, producing measurable ν-reduction.

### Quantitative Form
$$\phi_{\text{recovered}} \cdot \nu_{\text{recovered}} \approx \phi_{\text{healthy}} \cdot \nu_{\text{healthy}} \approx 1$$

Recovery = return toward φ·ν = 1 with φ and ν approaching equal values.

### What This Predicts That Competing Theories Don't

| Framework Prediction | Standard Clinical Prediction |
|---------------------|---------------------------|
| φ increase predicts proportional ν decrease | Symptom reduction follows independent trajectories |
| φ·ν product normalizes toward 1 during recovery | No predicted relationship between coherence and arousal |
| Treatment failure = system cannot generate sufficient φ | Treatment failure = insufficient exposure or processing |

### Proposed Protocol
- **Longitudinal study:** Trauma-exposed populations (some developing PTSD, some resilient)
- **φ-proxy:** Narrative coherence scales (Pennebaker), identity integration measures, meaning-making assessments
- **ν-proxy:** Cortisol baseline, hypervigilance indices (acoustic startle), heart rate variability
- **Treatment tracking:** Monitor both proxies through therapy (EMDR, CPT, prolonged exposure)
- **Analysis:** Test whether φ·ν product converges toward 1 during successful treatment

### Kill Criterion
If φ-proxy and ν-proxy change independently (correlation < 0.3) during treatment, the reciprocal-constraint prediction fails.

### Clinical Significance
If confirmed: a quantitative diagnostic metric (φ·ν deviation from 1) more precise than categorical DSM diagnosis, and a treatment-response predictor.

## PREDICTION 5: COGNITIVE-EMOTIONAL PRODUCT CONSERVATION

### The Claim
Cognitive load and emotional arousal trade off hyperbolically — their product is approximately conserved. This extends dual-process theory by specifying the FORM of the trade-off.

### Quantitative Form
$$\text{Cognitive-proxy} \times \text{Emotional-proxy} \approx C$$

### What This Predicts That Competing Theories Don't

| Framework Prediction | Dual-Process Theory Prediction |
|---------------------|-------------------------------|
| Product is approximately constant | Resources compete but no specific functional form |
| Trade-off is hyperbolic (small cost near balance, accelerating away) | Trade-off may be linear or concave |
| Quadratic penalty near optimum (from Taylor expansion of sin) | No prediction about penalty shape |

### Proposed Protocol
- **Paradigm:** Dual-task with graded cognitive load (n-back, 0-back through 3-back) and graded emotional induction (neutral through high-arousal stimuli)
- **Cognitive proxy:** Working memory accuracy, reaction time
- **Emotional proxy:** Skin conductance, self-reported arousal, amygdala BOLD signal
- **Analysis:** At each cognitive-load level, measure emotional-proxy. Test whether product is constant (hyperbolic) vs. linear trade-off.

### Kill Criterion
If a linear model fits the cognitive-emotional trade-off significantly better than a hyperbolic model, the specific prediction fails.

## PREDICTION 6: η = 0 EXCHANGES OUTPERFORM η > 0 EXCHANGES

### The Claim
Zero-extraction exchanges (both parties fully comprehend the transaction) produce measurably better outcomes than extractive exchanges, following the specific form B(η) = sin(2·arctan(1/(1+η))).

### Quantitative Form
$$B(\eta) = \sin\left(2 \arctan\left(\frac{1}{1 + \eta}\right)\right)$$

At η = 0, B = 1 (maximum). As η increases, B drops — with quadratic penalty near η = 0 (small extractions are almost free) and accelerating penalty for larger extractions.

### What This Predicts That Competing Theories Don't

| Framework Prediction | Behavioral Economics Prediction |
|---------------------|---------------------------------|
| Specific B(η) functional form | Information asymmetry reduces welfare (qualitative) |
| Quadratic penalty near η = 0 | Linear or threshold effects |
| Penalty shape follows sin(θ) | No predicted functional form |

### Proposed Protocol
- **Paradigm:** Behavioral economics experiment with graded information asymmetry
- **η-proxy:** Quantified information gap between parties (bits of hidden information)
- **Performance proxies:** Joint surplus, satisfaction, dispute rate, repeat-transaction rate
- **Manipulation:** 5+ levels of information asymmetry from η = 0 (full transparency) to η >> 0
- **Analysis:** Fit performance vs η to sin(θ) form, linear model, step function

### Kill Criterion
If performance degrades linearly (not quadratically) with η near zero, the specific geometric prediction fails.
