## FRAMEWORK CONTEXT FOR REVIEWERS

This document is part of the **Emergentism** framework. The following definitions are provided so you can evaluate the enclosed content without any other document.

### Key Definitions

- **Riemann Sphere (S²):** The one-point compactification of the complex plane C U {oo}, also denoted CP1.
- **Frame product:** F(z) = z * sigma(z) = 1 for all z in CP1 (Paper A).
- **{0, 1, oo}:** The frame triad: additive identity (0), multiplicative identity (1), compactification point (oo).
- **K(x):** Standard Kolmogorov complexity -- the length of the shortest program that outputs x on a fixed universal Turing machine.
- **K_sc(x):** Self-contained Kolmogorov complexity -- the length of the shortest description of x that includes its own dictionary (all symbol definitions), requiring no external interpreter.
- **Evidence Tiers:** [E] = established mathematics; [S] = structurally derived from [E]; [I] = interpretive mapping; [C] = conjecture with explicit kill criteria.

### What This Is NOT

This framework does not claim to replace algorithmic information theory. It introduces a variant (K_sc) that measures self-contained descriptive complexity and argues that the frame equation achieves the minimum among non-trivial self-referential descriptions. The standard results (Kolmogorov, Chaitin, Godel) are used as-is.

## INSTRUCTIONS FOR THE REVIEWER

**Reviewer profile:** Theoretical computer scientist or logician with expertise in algorithmic information theory (Kolmogorov complexity, Chaitin's incompleteness), Godel's incompleteness theorems, and self-referential formal systems. Familiarity with fixed-point theorems is valuable. **Focus:** This paper introduces K_sc (self-contained Kolmogorov complexity) and claims that the equation 1 = 0 * oo is K_sc-minimal among non-trivial self-referential descriptions. Evaluate: (1) whether K_sc is well-defined (the reliance on "universally available concepts" in the dictionary base may introduce vagueness); (2) whether the proof of K_sc-minimality is rigorous or rests on the unproven conjecture that 4 content symbols is the minimum for self-reference (the paper acknowledges this as [C]); (3) whether the self-referential closure is genuine (each symbol defined in terms of the others) or circular in a trivially vacuous way; (4) whether the minmax property (minimum length, maximum scope) is correctly argued; (5) whether the connection to Chaitin and Godel (Sections 4.1-4.3) correctly represents those results; and (6) whether "self-containment as self-grounding" is a meaningful ontological claim.

## KILL CRITERIA

This paper is **falsified** if any of the following is exhibited:

1. A shorter self-contained self-referential non-trivial description is exhibited (fewer than 4 content symbols, same self-referential closure, non-trivial generative scope).
2. K_sc is shown to be ill-defined (the concept of "universally available concepts" is vague enough to make K_sc non-computable in a way that standard K is not).
3. The self-containment of the frame equation is shown to require external stipulation (the terms do NOT mutually define each other without importing some additional assumption).
4. A non-trivial self-grounding structure is exhibited that does not have the form of the frame triad (nothing/everything/unit).
