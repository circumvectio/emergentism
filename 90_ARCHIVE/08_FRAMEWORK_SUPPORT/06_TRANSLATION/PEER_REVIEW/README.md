## Evidence Tiers

- **[A]/legacy [E] Established:** Textbook external results. If the reviewer disputes these, they dispute standard mathematics.
- **[S] Structural:** Derived from accepted external results via standard proof techniques. Verifiable.
- **[I] Interpretive:** Structural parallels and mappings. Compelling but not proved.
- **[C] Conjecture:** Speculative. Marked as such. Has explicit kill criteria.

Archive compatibility note: preserved packets may still use the older `[E]` label. Split it into `[A]` for external facts or `[B]` for repository/runtime receipts before quoting forward into live surfaces.
