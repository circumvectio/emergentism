# 10_SPRINT_GATES

Sprint-gating utilities that assemble digests, lineage, receipts, and other
bounded operating views for execution management.

The physical lane stays numbered as `10_SPRINT_GATES/`. A compatibility package
at `../sprint_gates/` preserves the documented `python -m sprint_gates <name>`
entrypoint without renaming this folder.

## What Belongs Here

- sprint composition and digest utilities
- lineage, pulse, tissue, politics, and receipt helpers
- shared sprint-gate support code

active management lane or make governance decisions by itself.
