# Evolutionary Doctrine (The Mesh)

> **Position: 80_DAC_FRAME > 02__BLUEPRINT > EVOLUTIONARY_DOCTRINE

**Purpose**: To ensure the entity adapts faster than the environment (The Red Queen Effect).
**Concept**: We are not a hierarchy; we are an **Ecological Mesh** of specialized agents.

**Rosetta boundary:** [I] This paper defines an evolutionary architecture for
agent coordination. [D] Red Queen, niche, mutation, and symbiosis language is
metaphor until supported by measured adaptation, incident, promotion, or
specialization receipts.

## 1. The Mesh Network
- **Decentralized**: Agents do not wait for a central "Brain". They act based on local signals.
- **Stigmergy**: Agents communicate by modifying the environment (The Filesystem).
    - *Example*: Agent A creates a `PIR.md`. Agent B sees the file and starts collecting. No direct message was sent.

## 2. Niche Specialization
- [I] **Generalists Die**: An agent trying to do everything fails at everything.
- **Specialists Survive**:
    - **Hunters**: Find bugs/targets.
    - **Builders**: Write code.
    - **Guards**: Run tests/security.
- [I] **The Charter**: Every agent must claim a specific "Ecological Niche" (Habitat) via a `SPECIALIZATION_CHARTER`.

## 3. The Red Queen (Technology)
- **The Hypothesis**: "It takes all the running you can do, to keep in the same place."
- [I] **Implication**: If our Tech Stack is static, it is rotting.
- [I] **Response**: Constant R&D. We must move technology from TRL 1 (Idea) to TRL 9 (Production) continuously.

## 4. Pattern Promotion (The Historian Loop)
- **Concept**: Pain is information. Failure is a mutation signal.
- **The Historian**: A background process that reviews `04_REALITY/INCIDENTS/` and `07_WORK/LEARNINGS.md`.
- **The Loop**:
    1.  **Detection**: Find recurring failure modes (e.g., "Ambiguous PIR" mentioned 3 times).
    2.  **Synthesis**: Draft a new **Invariant ($I$)** for `01_CONSTITUTION/KERNEL_INVARIANTS.md` or a new **Constraint** for `CONTRACT.md`.
    3.  **Mutation**: Present the draft to the Court of Owls (Phase 27) for verification.
    4.  **Activation**: Ratify and update the "Genetic Code" (Doctrine).

## 5. Symbiosis (Human-Agent)
- **Mutualism**: Humans provide Intent (DNA). Agents provide Labor (Cells).
- **Parasitism**: Avoid agents that consume tokens without producing artifacts.
- [I] **Syntropy**: The system must net-reduce entropy and net-increase coherence ($\Phi$) over time.
