# Influence Operations Doctrine

> **Holographic Briefing**
> **Position: 80_DAC_FRAME > 02__BLUEPRINT > INFLUENCE_OPS
> **Status**: Canonical ✅
> **Intent**: Define and operationalize Influence Operations Doctrine.

---


**Purpose**: To systematically urge the target audience towards a desired behavior using psychology and behavioral economics.
**Method**: Military PSYOP framework + Behavioral Economics "Nudges".

**Rosetta boundary:** [I] This paper is an ethical influence blueprint. It does
not authorize deception, false flags, coercion, targeted manipulation, unlawful
communications, or unapproved campaigns. [B] Behavior-change claims require
truthful message receipts, consented execution, and impact evidence.

## 1. Ethical Guardrails (The "White Hat" Standard)
- [I] **Truth**: All themes and messages must be factually true. No disinformation.
- [I] **Transparency**: The source must be identifiable (no "Black Ops" / false flag).
- [I] **The Newspaper Test**: If this campaign was leaked to the NYT, would we be ashamed? If yes, ABORT.

## 2. The Influence Cycle (PSYOP)
1.  **Target Audience Analysis (TAA)**: Understand the *Psychographics* (Fears, Desires, Biases).
2.  **Theme Selection**: What is the "Big Idea" that resonates? (e.g. "Security", "Freedom").
3.  **Message Production**: Create the artifacts (Ads, Emails, Docs).
4.  **Dissemination**: Deliver via the right channel (Value Chain).
5.  [B] **Impact Assessment**: Did behavior change?

## 3. Behavioral Economics (The Nudge)
- **Choice Architecture**: Organize the context in which people make decisions.
- **Defaults**: Most people stick with the default. Make it the right choice.
- **Friction**: Add friction to bad behaviors. Remove it from valid ones.
- **Social Proof**: "99% of developers use X."

## 4. Cialdini's 6 Principles
1.  **Reciprocity**: Give value first.
2.  **Scarcity**: "Only 3 seats left."
3.  **Authority**: "Dr. X recommends."
4.  **Consistency**: "You said you wanted Y, so here is Y."
5.  **Liking**: Be human/relatable.
6.  **Consensus**: Everyone is doing it.
