## Definition of Done (a folder is "done" when…)
1. Folder-level **verdict** assigned (🟢/🟡/🟠/🔴) + per-key-doc verdicts.
2. Demons **logged with file references** and an η-sign.
3. Each demon has a **staged, K2/K3-safe action** (no execution without signature).
4. **Coverage disclosed** (read-in-full vs scanned) — no silent caps.
5. Checklist updated; **artifacts committed to git** (working tree churns — commit = durable).
