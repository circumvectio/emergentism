## 7. Tier A review checks

Before committing any Tier A annotation pass:

```bash
# Scope: canonical Foundation homes only
git diff --cached --name-only | grep -Ev '^01_EMERGENTISM/(00_META|0[1-7]_[A-Z_]+)/' \
  && echo "SCOPE VIOLATION" \
  || echo "scope ok"

# Do not annotate compatibility stubs by accident
git diff --cached --name-only | rg '^01_EMERGENTISM/(00_|01_FORMAL_SYSTEM/|02_THE_DERIVATION/|03_THE_PAPERS/|00_THE_TRANSCENDENTAL_TRINITY/)' \
  && echo "STUB SURFACE TOUCHED" \
  || echo "stub surfaces untouched"

# Basic frontmatter presence in changed canonical markdown files
git diff --cached --name-only -- '*.md' \
  | rg '^01_EMERGENTISM/(00_META|0[1-7]_[A-Z_]+)/' \
  | xargs -I{} sh -c 'rg -q "^rosetta:" "{}" || echo "missing rosetta: {}"'
```

Do not require perfection in the first pass. Require: correct scope, no stub edits, no cross-lane touch, and reviewable frontmatter.
