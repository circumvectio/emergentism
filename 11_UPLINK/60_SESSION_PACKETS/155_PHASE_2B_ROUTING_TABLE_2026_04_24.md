## 0. Axiomatic guard

Folder discipline serves the work; the work does not serve folder discipline. If a file lives outside its ideal L-folder but readers can find it, the discipline is serving them. If a file is "perfectly placed" but nobody reads it, the placement is gatekeeping. The routing table below is a tool; the goal is compound readability across sessions.
