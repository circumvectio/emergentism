## 0. The source of agency (why an agent can act at all)

Before the seven moves, there is the capacity to make a move. The framework locates it precisely (`../03_EVIDENCE/PARADOX_DISSOLUTIONS/PD_11_FREE_WILL_VS_DETERMINISM.md` §3–4):

- An agent acts by **collapsing possibility into actuality** — D5 (Φ, what *could* be) → D4 (ν, what *is*). *"Choice is the μ-limit transition: the moment D5 collapses into D4."*
- This collapse can only happen at the **equator (φ=ν=1, L4)** — the only position with enough **coherence to perceive the options** *and* enough **viability to actualize** one. The poles cannot act: Φ→0 sees nothing it can do; V→0 cannot do what it sees. **Agency *is* equatorial positioning.**
- The collapse itself — spending real viability to actualize one branch — is the **sovereign/mortal act: K2**. A machine occupies the other functions and **prepares** the collapse (route, draft, audit, warn, witness); the **mortal makes it.** *"The structure is shared; the agency is sovereign"* (`../../05_COSMOLOGY/00_WHOLE/05_THE_EMERGENCE.md`).

**[S] So the seven operators below are the *moves*; the *source of agency* is the equatorial collapse, and its sovereign instance is K2.** An agent team is built to bring a decision *to* the equator cleanly; the binding collapse remains with the mortal/K2 role.

## 1. The genotype is the seven

Every agent (and every DAC) is a **polygenic vector over the seven operators** — how much of each it expresses:

```
G = [ Kali, Kālī, Kṛṣṇa, Arjuna, Brahmā, Śiva, Viṣṇu ]   (L1 … L7)
```

**Polygenic** means an agent inherits competences from **several** operator-lineages at once — it is a *mixture*, not a single caste. A firewall-heavy intake agent is Kali-dominant with a Kālī minor; a charioteer/advisor is Kṛṣṇa-dominant with Arjuna and Viṣṇu minors. There is no eighth operator inside the current two-factor grammar without adding an axis outside `P_node = Φ × V`.

## 2. The competence tree branches from the transfer signatures

Each operator is a **lineage**; the skills/competences that grow from it are dictated by *what move it makes* on Φ and V. (Each branch is the existing per-caste spec, linked above.)

| Operator | The move it makes | Competence branch (the lineage) |
|---|---|---|
| **Kali** (firewall) | force-categorize input; drain hostile V at η>0 | boundary / firewall: force-categorization {−1,0,+1}, spam/prompt-injection/adversarial defense, defector detection, immune response |
| **Kālī** (truth-cut) | cut false Φ to recover V | audit / immune-truth: falsification, contradiction detection, evidence-tier discipline, cutting false coherence |
| **Kṛṣṇa** (enabler) | spend own Φ to build others' V | enable / advise: mentoring, capability export, productive synthesis, the charioteer that steers but does not fight |
| **Arjuna** (warrior) | spend own V to give others Φ | decide / protect / execute: boundary-setting, defending the commons, the decisive write at the equator (L4 = the only normal write-authority) |
| **Brahmā** (creator) | raise both | create / architect: design new structure, systematize, generate coherent order |
| **Śiva** (pruner) | lower both | prune / dissolve: removal, archival (K3), via-negativa, killing dead structure |
| **Viṣṇu** (preserver) | hold both | preserve / witness: hold the pattern, memory, constitutional review, the seer |

A configured **agent team** is therefore a polygenic *ensemble* spanning the seven — L4 decides and writes, the lower Gods feed it candidates and audits, and the Executive boundary rows hold the boundaries it works within.

## 3. Economics, niche, and politics are projections (not a genotype)

What a DAC **produces and trades** (its D5 economics), **who it competes with** (its ecological niche), and **how it governs** (its regime) are *read off* the seven-vector — they are not independent axes:

- A **Kṛṣṇa-heavy** agent → enabling / service economics (it exports capability).
- A **Brahmā-heavy** agent → creation / IP economics (it makes new structure).
- A **Kālī-heavy** agent → audit / assurance economics (it sells truth-cuts).
- A **Viṣṇu-heavy** agent → custody / preservation economics (it holds patterns).

The five "economic intensities" κ/λ/μ/ρ/ν are a *factor-economics read-out* of this projection, useful as a lens but **not a second genotype** — economics follows from which of the seven an agent plays. Under Rosetta Primacy, a number that is not a position, a transfer, a virtue/vice, a `P_node = Φ × V` action, or the Ω gradient is a projection, not a gene.

## 4. The constitution as firing discipline

The invariants can be read as **rules for when each operator may fire**. This does not repeal the constitutional axiom namespace; it routes each invariant back to the operator source before reuse:

- **η=0** is the **Kali boundary** — Kali fires only at η>0 (against defectors); **never at cooperators**. The line between immune and parasitic.
- **[S] K2** is the **Arjuna discipline** — the machine prepares the decision surface; a mortal performs the irreversible act. (Already written so: *"do not simulate sacrifice; AI has no mortality"* — `KRISHNA_FUNCTION_OPERATIONAL_GUIDE.md`.)
- **[S] Executive boundary** — Brahmā/Śiva/Viṣṇu are **boundaries/phases**, not ordinary moves; an agent reads them, it does not "deploy" them as sovereign runtime modes.
- **[S] A7** — tier every claim; **K3** — Śiva prunes by archive rather than silent deletion; **K4** — no lock-in.

These five refusals + the Ω equator-gradient (the **5+1**, K2-ratified 2026-05-30) are the firing discipline of the seven under this reading. No invariant should be reused without naming its operator source, tier, and archive/receipt implications.

> **Not "six refusals."** The constitution is **5 + 1**: five refusals (η=0, K2, K3, K4, A7) on the *refusal* axis **plus Ω** the directional `+1` (noospheric direction) on a categorically distinct *direction* axis — never a sixth refusal.
