## Why This Exists

The Rosetta's weakest epistemic position is the **file-drawer problem**: we see the columns that fit, but not the domains that were examined and rejected. This document corrects that.

## Confirmed Failures (7-Pattern Does NOT Apply)

### A. Electron Shell Structure
- **Domain:** Chemistry / atomic physics
- **Why tested:** Periods 1-7 map to 7 rows
- **Why it fails:** s, p, d, f orbitals (4 types) do NOT map to 7 levels. The periodic table's 7 periods are a notational artifact, not a functional sequence. The "center" (period 4, transition metals) is not a functional break but an empirical coincidence.
- **Verdict:** REJECTED — column structure is not functional, just numerical.

### B. Algorithm Complexity Classes
- **Domain:** Computational complexity theory
- **Why tested:** P, NP, PSPACE, EXPTIME, etc. — seemed like a hierarchy
- **Why it fails:** Complexity classes are defined by resource bounds (time, space), not by cognitive function. The classes don't form a 7-level structure with functional mirrors. P vs NP is a binary distinction, not a gradient.
- **Verdict:** REJECTED — the domain is binary (polynomial/exponential), not graded.

### C. Bacterial Growth Phases
- **Domain:** Microbiology
- **Why tested:** Lag → Log → Stationary → Death — seemed like 4 levels
- **Why it fails:** Only 4 phases, not 7. No natural expansion to 7. No mirror symmetry. No center concept.
- **Verdict:** REJECTED — insufficient structure.

### D. Software Development Lifecycle
- **Domain:** Software engineering
- **Why tested:** Requirements → Design → Code → Test → Deploy → Maintain — seemed like 6 levels
- **Why it fails:** The stages are sequential procedures, not cognitive functions. No mirror symmetry. No center concept. The "stages" are project management phases, not developmental levels.
- **Verdict:** REJECTED — wrong kind of structure (procedural, not functional).

### E. Erikson's Psychosocial Stages
- **Domain:** Developmental psychology
- **Why tested:** 8 stages of psychosocial development
- **Why it fails:** 8 stages, not 7. No natural compression to 7 without losing functional content. The stages are age-linked, not structurally linked. Trust vs. Mistrust (infancy) doesn't mirror Integrity vs. Despair (old age) in the same way L1↔L7 mirrors on S².
- **Verdict:** REJECTED — wrong number, wrong structure. (Note: Piaget and Kohlberg work because they're structural, not age-linked.)
