# D_SERIES_DOMAINS

> Domain columns of the Rosetta — each file reads one discipline or tradition against the seven-level S² geometry.

**Status:** Active
**Register:** `[A]` for established source scales · `[S]` for column structure · `[I]` for the cross-domain mapping
**Routing:** [`AGENTS.md`](AGENTS.md) is the lane route card; [`CLAUDE.md`](CLAUDE.md) is the Claude-tools compatibility shim, not a second authority.

## What This Folder Is

The "columns" of the Rosetta Stone. Where `D_SERIES_ROWS/` develops one row across domains, this folder develops one domain across the seven rows. Each `D##_*` file is a single domain projection.

Domain mappings must distinguish the established source fact `[A]` from the Emergentism/Rosetta interpretation `[I]`. No domain analogy upgrades itself above `[I]` without independent support or a repository receipt. See [The Honest Position](../../../../02_EPISTEMOLOGY/01_EVIDENCE_TIERS/00_THE_HONEST_POSITION.md).
