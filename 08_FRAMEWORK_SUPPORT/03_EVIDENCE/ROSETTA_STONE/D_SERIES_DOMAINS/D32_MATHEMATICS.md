## SPHERE CORRECTION

```
φ = cot(θ/2)     ν = tan(θ/2)     P∞ = φ · ν = 1 (everywhere)

B = sin θ = 2/(φ + 1/φ) = 2/(ν + 1/ν)

At equator (θ = π/2):  φ = 1, ν = 1, B = 1
```
