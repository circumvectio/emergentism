physically admissible means, and an explicit value commitment, which action is
chosen?** It does not decide ontology, manufacture evidence, or derive the Good
from the reciprocal chart.

## Typed action boundary

```text
D5 possible: modeled alternatives, ranking, represented futures
D4 actual: embodied means, commitment, performed action, record

model + option field -> selection + commitment receipt
action + environment -> outcome receipt
both receipts -> Soul Loop update
```

Anticipated futures can influence present action through present models. That
claim needs no physical signal from the future. The physical light cone remains
bounded by spacetime and `c`; an option cone is the subset of admissible
histories an agent can model, rank, coordinate, and reach.
